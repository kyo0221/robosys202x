# robosys202x
ロボットシステム学の練習レポジトリ(PGKS)

# plusコマンド
![test](https://github.com/kyo0221/robosys202x/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す.

## 使い方
* 数値を入力したテキストファイルを実行時に参照することによりファイル内の数値を足し合わせた値が出力される.

## 必要なソフトウェア
* Python　3.7~3.10

## テスト環境
* Ubuntu

## テストの結果
